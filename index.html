用CSS实现元素内容居中

2019-02-21 21:30:04 | CSS

1.水平居中

##实现水平居中是最简单的，文本内容直接居中即可
div{
width:300px;
height:300px;
text-align:center;
border: 1px solid black;
}

2.垂直居中

##实现垂直居中只介绍比较简单的3种方法，毕竟本人只是菜鸟一枚(TmT)

##2.1 设置行高与元素高度一致(只适用于1行内容)
div{
width:300px;
height:300px;
line-height:300px;
border: 1px solid black;
}

##2.2 根据高度控制内边距
div{
width:300px;
height:300px;
padding-top:140px; //这里大家可自行更改数值使之居中
border: 1px solid black;
}

##2.3 css动画实现（将内容放入p标签中）
div{
position: absolute;
width: 300px;
height: 300px;
border: 1px solid red;

}
p{
position: absolute;
width: 150px;
top: 50%;
left:50%;
transform: translate(-50%,-50%);
border: 1px solid black;
}
