利用Jquery实现轮播图

2019-01-22 19:12:28 | jquery

同样先说一下整体思路：Jquery实现轮播图时，是所有图片叠放在一个位置，轮播时当前的图片显示，其他图片隐藏，这样做的一个好处就是在网页布局时，轮播图的位置只占一小块，不会影响其他布局（js原生代码实现时会占据所有图片的空间）

html代码

<div class="slider">
<ul>
<li><img src="images/1.jpg" width="500px" height='200px'></li>
<li><img src="images/2.jpg" width="500px" height='200px'></li>
<li><img src="images/3.jpg" width="500px" height='200px'></li>
<li><img src="images/4.jpg" width="500px" height='200px'></li>
</ul>

<ol>
<li></li>
<li></li>
<li></li>
<li></li>
</ol>
<!--箭头-->
<div class="arrow">
<span class="arrow-left">&lt;</span>
<span class="arrow-right">&gt;</span>
</div>
</div>

Jquery代码

1 自动播放

var index=0;
$('ol li').eq(index).css({'background':'red'}).siblings('li').css({'background':'#fff'});
var timeId=setInterval(play,3000);

function play(){
    index++;
    if(index==$('ul li').length){
        index=0;
}

  $('ul li').eq(index).fadeIn().siblings('ul li').fadeOut();
  $('ol li').eq(index).css({'background':'red'}).siblings('li').css({'background':'#fff'});
        }

2 左右箭头

var count = 0;

$(".arrow-right").click(function () {
  count++;
  if(count == $("ul li").length){
    count = 0;
  }
  //让count渐渐的显示，其他兄弟渐渐的隐藏
  $("ul li").eq(count).fadeIn().siblings("li").fadeOut();
  $('ol li').eq(count).css({'background':'red'}).siblings('li').css({'background':'#fff'});
});

$(".arrow-left").click(function () {
  count--;

  if(count == -1){
    count = $("ul li").length - 1;
  }
  //让count渐渐的显示，其他兄弟渐渐的隐藏
  $("ul li").eq(count).fadeIn().siblings("li").fadeOut();
  $('ol li').eq(count).css({'background':'red'}).siblings('li').css({'background':'#fff'});
})

3 进入元素停止定时器，离开开启

$('.slider').hover(function(){
      clearInterval(timeId);
    },function(){
       timeId=setInterval(play,3000);
    })

使用Jquery的好处就是Jquery帮我们封装好了一些函数，可以直接使用，不像js得自己封装，但个中原理都是一样的。
