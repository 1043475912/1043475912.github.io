利用三维变换实现轮播图

2019-01-23 21:59:08 | HTML5

同样再说一下整体思路：HTML5实现轮播图时，是所有图片叠放在一个位置，然后通过平移和旋转，形成立方体的4个面，通过旋转立方体实现图片轮播，这里主要熟悉三维变换，缺点就是只能轮播4个图片。

html代码

<div class="view">
<ul>
<!--一个li就是一个结构块,包括4个面-->
<li>
<!--每一个span是这个结构块中的某一个面，用来显示图片-->
<span></span>
<span></span>
<span></span>
<span></span>
</li>
</ul>
<div class="arrow">
<a class="pre">&lt;</a>
<a class="next">&gt;</a>
</div>
</div>

这里我只用了一个结构块，可以看到我们所见的图片轮播，也可以用多个结构块实现切割轮播

css代码

*{
    padding: 0;
    margin: 0;
}
.view{
    width: 560px;
    height: 300px;
    margin:100px auto;
    position: relative;
}
ul{
    width: 100%;
    height: 100%;
    list-style: none;
    /*transform: rotate3d(1,1,0,-30deg); 旋转才能看到立方体*/
    transform-style: preserve-3d;
}
ul > li{
    width: 100%;
    height: 100%;
    float: left;
    position: relative;
    transform-style: preserve-3d;
    /*添加过渡效果*/
    transition: transform 0.5s;
}
ul > li > span{
    width: 100%;
    height: 100%;
    position: absolute;
    left: 0;
    top: 0;
}
ul > li > span:nth-of-type(1) {
    background: url("");
    /*往z轴正方向偏移*/
    transform: translateZ(150px);
}
ul > li > span:nth-of-type(2){
    background: url("");
    transform: translateY(-150px) rotateX(90deg);
}
ul > li > span:nth-of-type(3){
    background: url("");
    transform: translateZ(-150px) rotateX(180deg);
}
ul > li > span:nth-of-type(4){
    background: url("");
    transform: translateY(150px) rotateX(-90deg);
}
.pre,.next{
    width: 60px;
    height: 60px;
    text-align: center;
    line-height:60px;
    position: absolute;
    top: 50%;
    font-size: 40px;
    transform: translate(0,-50%);
    background-color: rgba(0,0,0,0.5);
    color: #fff;
}
.arrow{
    width:560px;
    height:50px;
    display: none;
}
.view:hover .arrow{
    display: block;
}
.pre:hover,
.next:hover {
    cursor: pointer;
}
.pre{
    left: 0;
}
.next{
    right: 0;
}

用css3代码可以实现元素的移动和旋转，好处就是可以代替一部分js代码，所以这里的css代码多了点

Jquery代码

1 自动播放

var timeId=setInterval(play,3000);
var count=0;
function play(){
count++;

     $("li").each(function(key,value){ //0 1 2 3 4
              /*通过添加transform样式进行旋转*/
              $(this).css({
                  "transform":"rotateX("+(-count*90)+"deg)"
              });
          }); 
}

2 左右箭头

/*定义图片索引*/
var index=0;
/*添加节流阀 */
var flag=true;
/*下一张*/
$(".next").on("click",function(){
    if(flag==true){
        /*设置节流阀*/
        flag=false;
        index--;
        /*所谓下一张，就是将所有li元素围绕x旋转*/
        $("li").each(function(key,value){ //0 1 2 3 4
            /*通过添加transform样式进行旋转*/
            $(this).css({
                "transform":"rotateX("+(index*90)+"deg)"
            });
        });
        setTimeout(function(){
            flag=true;
        },1000);//延迟1秒用户才能再次点击，防止用户多次点击
    }
});

/*上一张*/
$(".pre").on("click",function(){
    if(flag==true){
        /*设置节流阀*/
        flag=false;
        index++;
        /*所谓下一张，就是将所有li元素围绕x旋转*/
        $("li").each(function(key,value){ //0 1 2 3 4
            /*通过添加transform样式进行旋转*/
            $(this).css({
                "transform":"rotateX("+(index*90)+"deg)"
            });
        });
        setTimeout(function(){
            flag=true;
        },1000);
    }
});

3 进入元素停止定时器，离开开启

$('.view').hover(function(){
  clearInterval(timeId);
},function(){
   timeId=setInterval(play,3000);
})
